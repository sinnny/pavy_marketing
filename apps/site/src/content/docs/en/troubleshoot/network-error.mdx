---
title: NETWORK_ERROR
description: An error occurred while communicating with the Pavy server.
---

## What does this mean?
The widget could not reach the Pavy server (`api.pavy.ai`) due to a loss of internet connection, DNS configuration issues, or a corporate firewall.

## How to verify?
1. Check if other websites are accessible.
2. Try to access `https://api.pavy.ai/health` in your browser address bar.

## How to fix?
1. **Check Connection:** Verify your internet connection.
2. **Firewall Settings:** If using a corporate network, ask your security administrator to whitelist the `*.pavy.ai` domain.
3. **Check VPN:** If a VPN is active, try disabling it and re-testing.

![Failed request example in DevTools Network tab](/images/troubleshoot/network-error-devtools.svg)

## Still stuck?
If the issue occurs only on specific devices or networks, you should check the proxy or security settings of those environments.
