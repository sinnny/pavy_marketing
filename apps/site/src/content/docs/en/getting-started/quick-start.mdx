Welcome to Pavy.ai! This guide will help you integrate our autonomous AI agent into your website.

<h2 id="api-key">1. Get your API Key</h2>

First, you'll need an API key from the [Pavy.ai Admin Dashboard](https://admin.pavy.ai). If you don't have an account yet, [sign up for free](https://admin.pavy.ai/signup).

<h2 id="script-tag">2. Add the Script Tag</h2>

Copy and paste the following script tag into your HTML's `<head>` or before the closing `</body>` tag.

```html
<script 
  src="https://cdn.pavy.ai/widget/v1/pavy.js" 
  data-api-key="YOUR_API_KEY"
  async
></script>
```

<h2 id="verify">3. Verify Installation</h2>

Once you've added the script, refresh your website. You should see the Pavy.ai chat bubble appear in the bottom right corner.
