---
title: APIキーの管理
description: Pavy.aiのAPIキーを確認・ローテーション・安全に管理する方法。
---

APIキーは、Pavy.aiがウェブサイトを認証し、対応するナレッジベースに紐づけるための主要な手段です。

<h2 id="finding">APIキーの確認方法</h2>

1. [Pavy.ai管理ダッシュボード](https://admin.pavy.ai)にログインします。
2. **設定 > ウィジェット構成**に移動します。
3. 「連携(Integration)」セクションに公開APIキーが表示されます。

<h2 id="security">セキュリティのベストプラクティス</h2>

- **公開前提:** スクリプトタグで使用するAPIキーは公開キーです。クライアントサイドのコードで利用される前提で設計されています。
- **ドメインホワイトリスト:** セキュリティを高めるため、管理ダッシュボードでドメインをホワイトリストに登録することを推奨します。これにより、他サイトからのAPIキー利用を防げます。
- **ローテーション:** APIキーが漏えいした可能性がある場合は、ダッシュボードからローテーションできます。ローテーション後は、ウェブサイトのスクリプトタグを直ちに更新してください。

<h2 id="env-vars">環境変数</h2>

ReactやNext.jsなどのフレームワークを使用している場合、APIキーを環境変数で管理することを推奨します。

```bash
VITE_PAVY_API_KEY=your_api_key_here
```

コードでは次のように参照します：

```javascript
const apiKey = import.meta.env.VITE_PAVY_API_KEY;
```
