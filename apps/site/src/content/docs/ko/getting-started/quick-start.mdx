---

Pavy.ai에 오신 것을 환영합니다! 이 가이드는 자율형 AI 에이전트를 여러분의 웹사이트에 연동하는 방법을 안내합니다.

<h2 id="api-key">1. API 키 발급받기</h2>

먼저 [Pavy.ai 관리자 대시보드](https://admin.pavy.ai)에서 API 키를 발급받으세요. 아직 계정이 없다면 [무료로 가입](https://admin.pavy.ai/signup)할 수 있습니다.

<h2 id="script-tag">2. 스크립트 태그 추가하기</h2>

아래 스크립트 태그를 복사해 HTML의 `<head>` 안 또는 `</body>` 닫는 태그 바로 앞에 붙여넣으세요.

```html
<script 
  src="https://cdn.pavy.ai/widget/v1/pavy.js" 
  data-api-key="YOUR_API_KEY"
  async
></script>
```

<h2 id="verify">3. 설치 확인하기</h2>

스크립트를 추가한 후 웹사이트를 새로고침하면 오른쪽 하단에 Pavy.ai 채팅 버블이 나타납니다.
