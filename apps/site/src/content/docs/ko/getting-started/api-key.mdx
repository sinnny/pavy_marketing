API 키는 Pavy.ai가 여러분의 웹사이트를 인증하고 해당 지식 베이스와 연결하기 위한 핵심 수단입니다.

<h2 id="finding">API 키 확인하기</h2>

1. [Pavy.ai 관리자 대시보드](https://admin.pavy.ai)에 로그인합니다.
2. **설정 > 위젯 구성** 메뉴로 이동합니다.
3. "연동(Integration)" 섹션에서 공개 API 키를 확인할 수 있습니다.

<h2 id="security">보안 권장 사항</h2>

- **공개 키 성격:** 스크립트 태그에 사용되는 API 키는 공개 키입니다. 클라이언트 측 코드에서 사용하도록 설계되었습니다.
- **도메인 화이트리스트:** 보안 강화를 위해 관리자 대시보드에서 도메인을 화이트리스트에 등록하는 것을 권장합니다. 이렇게 하면 다른 사이트에서 여러분의 API 키를 사용할 수 없습니다.
- **키 교체(Rotation):** API 키가 유출되었을 가능성이 있다면 대시보드에서 교체할 수 있습니다. 교체한 뒤에는 웹사이트의 스크립트 태그를 즉시 업데이트해야 합니다.

<h2 id="env-vars">환경 변수 사용</h2>

React나 Next.js 같은 프레임워크를 사용한다면, API 키를 환경 변수로 관리하는 것을 권장합니다.

```bash
VITE_PAVY_API_KEY=your_api_key_here
```

코드에서 다음과 같이 참조합니다:

```javascript
const apiKey = import.meta.env.VITE_PAVY_API_KEY;
```
