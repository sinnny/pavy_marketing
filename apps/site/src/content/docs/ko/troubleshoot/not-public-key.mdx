---
title: NOT_PUBLIC_KEY
description: 위젯 설치에는 '퍼블릭 키(Public Key)'를 사용해야 합니다.
---

## What does this mean?
Pavy에는 두 종류의 키가 있습니다.
- **Public Key (`pk_`로 시작):** 웹사이트 위젯 설치용. 보안상 외부에 노출되어도 안전합니다.
- **Secret Key (`sk_`로 시작):** 서버 간 통신(백엔드 API 호출)용. 절대로 웹사이트 소스 코드에 노출해서는 안 됩니다.

현재 위젯 설치 스크립트에 `sk_`로 시작하는 시크릿 키가 사용되어 보안 정책상 차단되었습니다.

## How to verify?
웹사이트 소스 코드의 `data-api-key` 값이 `sk_`로 시작하는지 확인하세요.

## How to fix?
1. [Pavy 어드민](https://admin.pavy.ai)의 **설정 > API 키** 메뉴로 이동합니다.
2. **Public Key** 섹션에 있는 `pk_`로 시작하는 키를 복사합니다.
3. 웹사이트의 스크립트 태그 내 `data-api-key` 값을 복사한 퍼블릭 키로 교체합니다.
