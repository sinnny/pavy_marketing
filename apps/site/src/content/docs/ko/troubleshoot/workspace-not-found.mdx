title: WORKSPACE_NOT_FOUND
description: 연결된 워크스페이스를 찾을 수 없거나 비활성화되었습니다.
---

## What does this mean?
입력한 API 키에 해당하는 워크스페이스가 시스템에 존재하지 않거나, 계정 정지 또는 삭제로 인해 비활성화된 상태입니다.

## How to verify?
사용 중인 API 키가 올바른 계정에서 발급된 것인지 확인하세요.

## How to fix?
1. Pavy 어드민에 정상적으로 로그인되는지 확인합니다.
2. 계정 요금제 결제 상태를 확인하세요. 미납으로 인해 서비스가 일시 중단되었을 수 있습니다.
3. 새로운 워크스페이스를 생성했다면, 이전 워크스페이스의 키를 그대로 사용하고 있지는 않은지 확인하세요.

## Still stuck?
계정 상태에 이상이 없음에도 이 에러가 발생한다면 [고객 지원팀](mailto:support@pavy.ai)으로 문의해 주세요.
