---
title: INVALID_API_KEY
description: 제공된 API 키가 유효하지 않거나 만료되었습니다.
---

## What does this mean?
위젯 SDK 초기화 과정에서 사용된 API 키가 Pavy 시스템에서 확인되지 않거나, 비활성화된 경우입니다. API 키가 정확하지 않으면 위젯이 서버로부터 설정을 불러올 수 없습니다.

## How to verify?
1. 웹사이트 소스 코드의 스크립트 태그 내 `data-api-key` 값을 확인합니다.
2. 예: `<script src="..." data-api-key="pk_live_...">`

## How to fix?
1. [Pavy 어드민](https://admin.pavy.ai)에 로그인합니다.
2. **설정 > API 키** 메뉴로 이동합니다.
3. 현재 사용 중인 키가 목록에 있는지, '활성' 상태인지 확인합니다.
4. 만약 키가 다르다면, 어드민의 키를 복사하여 웹사이트 소스 코드에 붙여넣으세요.

## Still stuck?
키가 정확함에도 문제가 지속된다면 [고객 지원팀](mailto:support@pavy.ai)으로 문의해 주세요.
