---
title: ORIGIN_NOT_ALLOWED
description: 현재 도메인이 허용된 도메인 목록에 등록되어 있지 않습니다.
---

## 어떤 의미인가요?
보안을 위해 Pavy 위젯은 사전에 등록된 도메인(Origin)에서만 동작합니다. 다른 사이트에서 귀하의 API 키를 도용하는 것을 방지하기 위한 조치입니다.

## 확인 방법
브라우저 콘솔 에러 메시지에 표시된 `Origin: https://...` 주소가 귀하의 실제 웹사이트 주소와 일치하는지 확인하세요.

## 해결 방법
1. [Pavy 어드민](https://admin.pavy.ai)의 **설정 > 도메인 관리** 메뉴로 이동합니다.
2. 현재 위젯을 설치하려는 웹사이트의 도메인을 추가합니다 (예: `https://my-shop.com`).
3. **와일드카드 지원:** 모든 서브도메인을 허용하려면 `*.my-shop.com` 형식을 사용하세요.
4. 저장 후 약 1~2분 뒤에 웹사이트를 새로고침하여 확인합니다.

![어드민 > 설정 > 도메인 관리 캡처](/images/troubleshoot/origin-not-allowed-admin.svg)

## 여전히 해결되지 않나요?
도메인을 등록했음에도 문제가 지속된다면, 브라우저의 캐시를 삭제하거나 시크릿 창에서 다시 시도해 보세요.
