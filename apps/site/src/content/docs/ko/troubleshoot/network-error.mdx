---
title: NETWORK_ERROR
description: Pavy 서버와의 통신 중 네트워크 오류가 발생했습니다.
---

## 어떤 의미인가요?
위젯 로드 중 인터넷 연결이 끊기거나, DNS 설정 문제, 또는 사내 방화벽으로 인해 Pavy 서버(`api.pavy.ai`)에 접근하지 못하는 상태입니다.

## 확인 방법
1. 다른 웹사이트는 정상적으로 접속되는지 확인합니다.
2. 브라우저 주소창에 `https://api.pavy.ai/health`를 입력하여 접속되는지 확인하세요.

## 해결 방법
1. **네트워크 확인:** 인터넷 연결 상태를 점검합니다.
2. **방화벽 설정:** 사내 네트워크를 사용 중이라면 보안 담당자에게 `*.pavy.ai` 도메인에 대한 통신 허용을 요청하세요.
3. **VPN 확인:** VPN이 활성화되어 있다면 종료 후 다시 시도해 보세요.

![DevTools Network 탭에서 실패한 요청 예시 캡처](/images/troubleshoot/network-error-devtools.svg)

## 여전히 해결되지 않나요?
특정 기기나 네트워크에서만 발생한다면 해당 환경의 프록시 또는 보안 설정을 점검해야 합니다.
