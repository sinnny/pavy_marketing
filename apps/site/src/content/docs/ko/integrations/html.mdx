title: HTML / Vanilla JS 연동
description: 표준 HTML과 JavaScript만으로 Pavy.ai를 어떤 웹사이트에든 연동하세요.
---

Pavy.ai는 플랫폼에 구애받지 않고 간단하게 연동할 수 있습니다. 스크립트 태그 하나만 추가하면 방문자에게 AI 기반 쇼핑 어시스턴트를 제공할 수 있습니다.

## 사전 준비

시작하기 전에 다음 사항을 준비해 주세요:
1. **Pavy.ai 계정** — 계정이 없다면 [이곳에서 가입](https://pavy.ai/signup)하세요.
2. **API 키** — [대시보드 > 설정 > API 키](https://pavy.ai/dashboard/settings)에서 확인할 수 있습니다.

## 설치 단계

아래 순서에 따라 Pavy.ai 위젯을 웹사이트에 추가하세요:

1. 아래 스크립트 태그를 복사합니다.
2. 웹사이트의 메인 HTML 파일(보통 `index.html` 또는 공통 푸터 템플릿)을 엽니다.
3. `</body>` 닫는 태그 바로 앞에 스크립트 태그를 붙여넣습니다.
4. `YOUR_API_KEY`를 실제 API 키로 교체합니다.

```html
<script 
  src="https://cdn.pavy.ai/widget/v1/pavy.js" 
  data-api-key="YOUR_API_KEY"
  async
></script>
```

## 설정 옵션

스크립트가 로드되기 *전에* `window.PavyConfig` 객체를 정의해 위젯 동작을 커스터마이즈할 수 있습니다.

| 옵션 | 타입 | 설명 |
| :--- | :--- | :--- |
| `theme` | `string` | 'light' 또는 'dark'. 기본값은 'light'. |
| `primaryColor` | `string` | 메인 브랜드 컬러의 HEX 코드 (예: '#6366f1'). |
| `position` | `string` | 'right' 또는 'left'. 기본값은 'right'. |
| `user` | `object` | `id`, `name`, `email`을 포함하는 선택적 객체. |

고급 설정 예시:

```html
<script>
  window.PavyConfig = {
    theme: 'dark',
    primaryColor: '#6366f1',
    position: 'right',
    user: {
      id: 'user_123',
      name: 'Jane Doe',
      email: 'jane@example.com'
    }
  };
</script>
<script src="https://cdn.pavy.ai/widget/v1/pavy.js" data-api-key="YOUR_API_KEY" async></script>
```

## 설치 확인

스크립트를 추가하고 변경 사항을 저장한 뒤:
1. 브라우저에서 웹사이트를 새로고침합니다.
2. 하단 모서리(기본은 오른쪽 아래)에 Pavy.ai 채팅 버블이 보여야 합니다.
3. 버블을 클릭해 채팅 창이 정상적으로 열리는지 확인합니다.

## 상품 컨텍스트 설정

사용자가 보고 있는 상품을 Pavy.ai가 인식하도록 하려면, 상품 상세 페이지에 상품 컨텍스트를 제공해야 합니다.

```html
<script>
  window.PavyConfig = {
    product: {
      id: 'sku_123',
      title: 'Premium Wireless Headphones',
      price: 199.99,
      currency: 'USD',
      url: window.location.href
    }
  };
</script>
```

## 문제 해결

| 문제 | 해결 방법 |
| :--- | :--- |
| **위젯이 나타나지 않음** | API 키가 올바른지, 스크립트 URL이 접근 가능한지 다시 확인하세요. 광고 차단기가 스크립트 로드를 막고 있지 않은지도 확인하세요. |
| **채팅 버블이 콘텐츠를 가림** | `PavyConfig`의 `position`을 변경하거나, 겹치는 요소의 z-index를 CSS로 조정하세요. |
| **스타일이 어긋남** | 위젯은 스타일 간섭을 막기 위해 Shadow DOM을 사용하지만, 사이트의 전역 CSS가 `iframe`이나 `pavy-widget` 요소를 과도하게 타게팅하지 않도록 주의하세요. |
