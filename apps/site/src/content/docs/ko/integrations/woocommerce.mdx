---
title: WooCommerce 연동
description: WordPress/WooCommerce 사이트에 Pavy.ai를 몇 분 만에 추가하세요.
---

Pavy.ai를 WooCommerce에 연동하면 WordPress 사이트에서 바로 즉각적인 고객 지원과 상품 추천을 제공할 수 있습니다.

## 사전 준비

시작하기 전에 다음 사항을 준비해 주세요:
1. **Pavy.ai 계정**.
2. WordPress 대시보드의 **관리자 권한**.
3. **Pavy API 키**.

## 설치 단계

### 방법 1: functions.php 사용 (권장)

1. WordPress 대시보드에 로그인합니다.
2. **외모(Appearance) > 테마 파일 편집기**로 이동합니다.
3. 활성화된 (자식) 테마를 선택하고 `functions.php` 파일을 엽니다.
4. 파일 맨 아래에 다음 코드를 추가합니다:

```php
add_action('wp_footer', 'add_pavy_ai_widget');
function add_pavy_ai_widget() {
    ?>
    <script 
      src="https://cdn.pavy.ai/widget/v1/pavy.js" 
      data-api-key="YOUR_API_KEY"
      async
    ></script>
    <?php
}
```

5. **파일 업데이트**를 클릭합니다.

### 방법 2: Code Snippets 플러그인 사용

테마 파일 편집을 피하고 싶다면 "WPCode" 같은 플러그인을 사용하세요:
1. **WPCode**를 설치하고 활성화합니다.
2. **Code Snippets > Header & Footer**로 이동합니다.
3. **Footer** 섹션에 Pavy.ai 스크립트 태그를 붙여넣습니다.
4. **변경 사항 저장**을 클릭합니다.

## 설정 옵션

WordPress 사용자 데이터를 Pavy에 전달해 더 개인화된 경험을 제공할 수 있습니다:

| 옵션 | PHP 코드 | 설명 |
| :--- | :--- | :--- |
| `user.id` | `get_current_user_id()` | 로그인한 사용자의 ID. |
| `user.name` | `wp_get_current_user()->display_name` | 로그인한 사용자의 이름. |
| `theme` | `'light'` 또는 `'dark'` | 위젯의 비주얼 테마. |

## 설치 확인

1. 웹사이트 홈페이지를 방문합니다.
2. 하단 모서리에서 Pavy.ai 채팅 버블을 찾아보세요.
3. "안녕하세요" 같은 메시지를 보내 연결이 활성화되어 있는지 확인합니다.

## 상품 컨텍스트 설정

고객이 보고 있는 상품에 대한 컨텍스트를 Pavy에 제공하려면, `functions.php`에 다음 스니펫을 추가하세요:

```php
add_action('wp_head', 'pavy_product_context');
function pavy_product_context() {
    if ( is_product() ) {
        global $product;
        ?>
        <script>
          window.PavyConfig = {
            product: {
              id: '<?php echo $product->get_id(); ?>',
              title: '<?php echo esc_js($product->get_name()); ?>',
              price: '<?php echo $product->get_price(); ?>',
              currency: '<?php echo get_woocommerce_currency(); ?>',
              sku: '<?php echo $product->get_sku(); ?>'
            }
          };
        </script>
        <?php
    }
}
```

## 문제 해결

| 문제 | 해결 방법 |
| :--- | :--- |
| **위젯이 나타나지 않음** | WP Rocket, W3 Total Cache 같은 캐시 플러그인을 사용 중이라면 스크립트 추가 후 사이트 캐시를 비워주세요. |
| **스크립트가 로드되지 않음** | `functions.php`에 문법 오류가 없는지 확인하세요. 사이트가 백지 상태가 되면 FTP/파일 관리자로 추가한 코드를 제거하세요. |
| **상품 정보가 부정확함** | 테마가 `is_product()` 검사를 우회하는 커스텀 상품 템플릿을 쓰고 있는지 확인하세요. |
